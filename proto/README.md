# Contrato gRPC — en preparación

Aquí vivirá `flux.proto`: el contrato para **plugins-servicio** (procesos externos en
Go, PHP vía FrankenPHP, o cualquier lenguaje con gRPC), equivalente a la superficie
`flux` del motor embebido.

Aún no publicado. Mientras tanto, el runtime disponible es el de scripts
(JS/TypeScript) — ver [`spec/`](../spec/).
